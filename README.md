## 📁 目录结构

```
my-articles/
├── 01-AI基础科普/           ← 仅存放文章 .md
├── 02-提示词与工具教程/
├── 03-Agent与前沿技术/
├── 04-产品测评/
├── 05-行业动态/
├── 引用的图片/              ← 所有文章配图，按「分类/文章」两层存放
│   ├── 01-AI基础科普/<文章名>/01.png ...
│   └── ...
└── README.md              ← 本文件（文章总索引）
```

分类目录下只放文章正文 `*.md`；每篇文章引用的配图统一收纳在根目录的 `引用的图片/` 下，按「分类 / 文章」两层组织。
